<h2>Setup install</h2>
Have python 3.8+ and VScode downloaded
<h2>Installation</h2>
Run:<br>
<code>pip install -r requirements.txt</code><br>
in terminal for requirements
<h2>Run</h2>
In terminal run:<br>
<code>python manage.py runserver </code> <br>
Then copy the link into the browser and run.
